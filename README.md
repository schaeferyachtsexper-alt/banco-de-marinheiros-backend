# Banco de Marinheiros

Plataforma náutica profissional para conectar tripulações qualificadas (marinheiros, capitães, ajudantes etc.) a embarcações de médio e alto padrão.

Este repositório contém:

- Uma base de front-end em Next.js 15 + Tailwind.
- Componentes reutilizáveis em estilo shadcn/ui (Button, Card).
- Páginas iniciais para a landing page (app/page.tsx).
- Pasta `docs/` com rascunhos de visão de produto, business plan e documento para investidores.

## Como rodar o projeto

1. Instale as dependências:

```bash
npm install
# ou
yarn
```

2. Rode o ambiente de desenvolvimento:

```bash
npm run dev
```

3. Acesse em http://localhost:3000
